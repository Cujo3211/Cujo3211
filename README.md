# Halo MCC Tracker

Netlify publish directory: `public`

Main tracker file: `public/index.html`

Main data file: `public/data/campaign_times.json`

Database tab default password: `117`

When you edit times in the live tracker, use **SAVE CHANGES + DOWNLOAD JSON** and replace `public/data/campaign_times.json` in your local folder. Browser edits do not automatically change your offline files or Netlify deploy source.
